# Portable Demo Pack

This sample pack groups deterministic macros and assets that can be shipped with a portable build.

Included macros:
- `../../samples/focus_and_paste.json`
- `../../samples/image_click_retry.json`
- `../../samples/upload_and_submit.json`

Required snippets:
- `assets/snippets/hello.txt`
- `assets/snippets/prompt_01.txt`
- `assets/snippets/signature.txt`

Required templates:
- `assets/templates/calc_equals.png`
- `assets/templates/done_icon.png`
- `assets/templates/upload_button.png`

Recommended settings:
- Use `config/app.json` for portable deployments.
- Keep the emergency stop hotkey at `Ctrl+Alt+Pause`.
